Kmer Counting K-mer counting is one of the simplest yet very powerful types of DNA analysis. We can represent any DNA sequence as a string of four letters: [A,C,G,T]. A k-mer of size k is a substring of k letters. For a sequence of length l we can enumerate (l - k + 1) k-mers. For instance, if we consider the DNA sequence and pick k = 3 we can enumerate the following 3-mers: [ACT, CTT, TTG, TGA, GAC, ACT, CTT]. The task is to return the number of times a k-mer appears within a sequence. C++
